# Green-Branding-and-Gen-Z-Purchase-Behavior--Data-Processing
Project Overview

This repository contains the full analytical pipeline for the research study:

“Green Branding and Gen Z Purchase Behavior: A Multi-Level Empirical Analysis of Consumer Perception and Brand-Level Sustainability Positioning in India.”

The study examines whether green branding independently influences sustainable purchase intention among Indian Generation Z consumers (16–26 years), or whether identity-based psychological factors play a stronger role.

Using primary survey data (N = 141), the analysis integrates:

Environmental Identity

Green Branding Perception

Social Influence

Price Sensitivity

Sustainable Purchase Intention (Binary DV)

The study employs both Ordinary Least Squares (Linear Probability Model) and Logistic Regression to test the robustness of findings.

-> Methods Implemented

The repository includes:

Data cleaning and preprocessing

Composite score construction

Reliability Analysis (Cronbach’s Alpha)

Descriptive Statistics

Pearson Correlation Matrix

OLS Regression

Logistic Regression

Regression Diagnostics

Visualization (distribution plots, correlation heatmap, residual analysis)

All analyses were conducted using Python.

-> Key Finding

Environmental Identity emerged as the only statistically significant predictor of sustainable purchase intention. Green Branding, Social Influence, and Price Sensitivity did not independently influence purchase intention once identity was controlled.

This suggests sustainable consumption among Indian Gen Z consumers is primarily identity-driven rather than branding-driven.
